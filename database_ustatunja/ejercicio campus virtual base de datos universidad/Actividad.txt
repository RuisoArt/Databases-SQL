Ejercicio No. 1: Descubrir los conceptos básicos de una base de datos 

Crear un archivo en excel con las hojas de cálculo necesarias para representar la estructura de administrativa - académica de la Universidad:

1.Nombrar cada hoja de cálculo de la siguiente manera:
Seccionales
Divisiones
Facultades
Programas
Pensum
Asignaturas
Estudiantes
2. Identificar Cuáles pueden hacer falta

3. Identificar para cada elemento las características que permitan hacer una descripción del mismo, e incluirlas como encabezado de cada columna dentro de cada hoja de cálculo según corresponda

4. Facilitar el ingreso de datos utilizando las funciones: buscarv (o consultav) y validación de datos

5. Asociar los conceptos de Entidad, Atributo, Instancia, Llave primaria y llave foránea con los diferentes elementos utilizados en el ejercicio
